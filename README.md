# Markdown Transfer

一个简洁优雅的 Markdown 转富文本工具。在浏览器中完成所有转换，无需服务器，保护您的隐私。

## 功能

- **实时预览** — 左侧输入 Markdown，右侧即时显示富文本效果
- **一键复制** — 点击「复制富文本」按钮，直接粘贴到 Word、WPS 等文档中
- **手动复制** — 也可以直接选中预览区域内容复制
- **客户端处理** — 所有转换在浏览器本地完成，数据不上传服务器

## 支持的 Markdown 语法

- 标题（H1-H6）
- 粗体、斜体
- 有序列表、无序列表
- 代码块与行内代码
- 引用块
- 链接
- 表格

## 使用

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 技术栈

- [Vite](https://vitejs.dev/) — 构建工具
- [marked](https://marked.js.org/) — Markdown 解析

## 浏览器支持

- Chrome / Edge / Firefox / Safari 最新版本
- 复制功能需要浏览器支持 Clipboard API
